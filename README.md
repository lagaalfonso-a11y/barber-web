# Barbería Web

Página web moderna para gestión de barbería con:
- Interfaz principal con carrusel y servicios
- Panel administrativo
- Gestión de citas
- Galería de servicios

## Estructura

```
├── index.html         # Página principal
├── admin.html         # Panel administrativo
├── config.js          # Configuración
├── css/               # Estilos
│   ├── admin.css
│   ├── global.css
│   ├── responsive.css
│   ├── servicios.css
│   ├── style.css
│   ├── variables.css
│   └── ...
├── js/                # JavaScript
│   ├── admin.js
│   ├── api-interactions.js
│   ├── config.js
│   ├── functions.js
│   ├── galeria.js
│   └── ...
└── README.md
```

## Cómo usar

1. Abre `index.html` en tu navegador
2. Para acceder al panel admin, ve a `admin.html`
3. Personaliza la configuración en `config.js`
